Manual

Como cadastrar filmes e séries

1. Abra a aba Filmes ou Séries.

2. Digite o nome do título.

3. Preencha ou confirme o gênero.

4. Escreva a sinopse, se quiser.

5. Defina a classificação por estrelas. Se tocar na mesma estrela de novo, a nota é removida.

6. Escolha o status: pendente, quero assistir ou assistido.

7. Toque em Salvar.

Como usar a TMDB API Key

1. Salve sua TMDB API Key nesta tela.

2. Depois volte para Filmes ou Séries.

3. Digite o nome do título e toque em Buscar na API.

4. O app pode preencher automaticamente nome, gênero, sinopse e capa.

Cadastro no TMDB

Para criar sua chave da API, abra o site oficial do TMDB e faça seu cadastro.

Depois de entrar na sua conta, gere sua API Key e cole nesta tela.

Link direto:

https://www.themoviedb.org/signup
